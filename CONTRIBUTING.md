# Contributing to AutonomyHQ

AutonomyHQ is currently maintained by a solo developer. Contributions, suggestions, and bug reports are always welcome.

## How to contribute

1. Clone the repository.
2. Read the README to understand the project architecture and how to run it locally.
3. Create a new branch:

   ```bash
   git checkout -b my-change
   ```

4. Test your changes locally and make sure everything works as expected.
5. Commit your changes, push your branch, and open a pull request explaining what you changed and why.

## Please note

Do not commit secrets. Files such as `.env`, `.pem`, and private keys are intentionally ignored.

## Questions

If you have any questions or run into an issue, open a GitHub issue or email **autonomyhq00@gmail.com**.
