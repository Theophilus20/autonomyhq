# Security Policy

AutonomyHQ runs on Casper Testnet. It uses test tokens only and holds no real funds.

## Reporting a vulnerability

If you find a security issue, please email autonomyhq00@gmail.com rather than opening a public issue. I will respond as soon as I can.

## Known limitation

The hosted recorder signs transactions with a single operator key intended for testnet demonstration only. This is documented in the README and is not suitable for production or mainnet use.
